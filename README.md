# FuelWise - First Spring Boot Project

FuelWise is a beginner-friendly Spring Boot REST API created to learn backend development concepts while solving a practical problem.

The application compares ethanol-blended fuel with regular fuel by analyzing:

- Fuel mileage
- Fuel price
- Cost per kilometer
- Efficiency difference
- Savings or additional expense

Users can enter mileage and fuel costs for different fuel types, and the API returns a detailed comparison to help determine the most economical choice.

## Tech Stack

- Java 21
- Spring Boot
- Maven
- REST APIs
- JSON Request/Response

## Learning Objectives

- Spring Boot fundamentals
- Dependency Injection
- REST Controllers
- Service Layer Architecture
- Request and Response DTOs
- Maven Project Structure
- API Testing with Postman
